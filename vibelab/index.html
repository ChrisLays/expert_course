Chaque étoile est une compétence de vibe coder : une mini-leçon, puis une mission à accomplir dans Claude (claude.ai ou Claude Code). Validez vos missions pour allumer la constellation — VibeLab forge les prompts, Claude fait le travail, vous comprenez ce qui se passe.

Fondations d'une app L'art du prompt Outils Claude ◌ à explorer  ·  ● mission validée

La Forge fabrique des prompts experts prêts à coller dans Claude ou Claude Code. Décrivez votre situation, générez, copiez — et observez la différence avec un prompt improvisé. C'est aussi comme ça qu'on apprend à mieux prompter.

Comment ça marche : VibeLab n'embarque aucune IA et n'envoie rien nulle part — votre progression vit dans ce navigateur (localStorage). Les missions et les commandes produisent des prompts que vous exécutez vous-même dans Claude (claude.ai) ou Claude Code. L'objectif : acquérir les réflexes et le vocabulaire d'un développeur web… en laissant le code à Claude.
