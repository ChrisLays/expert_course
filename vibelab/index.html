26 étoiles-compétences. Le cursus web monte en 5 orbites — des fondations (N1) au niveau d'un développeur junior aux compétences solides (N5). Chaque étoile : une leçon dense, une mission à exécuter dans Claude, une validation. Les branches Prompt et Outils Claude complètent l'entraînement.

La Forge fabrique des prompts experts prêts à coller dans Claude ou Claude Code. Décrivez votre situation, forgez, copiez. Les commandes marquées CC méritent d'être installées en vraies commandes slash dans votre projet — le Codex explique comment.

La base de référence du vibe coder : l'outillage Claude réel et complet — commandes slash intégrées, CLI, raccourcis Mac, hooks, sous-agents, skills, mémoire, MCP, Artifacts, Projects. Cherchez, dépliez, apprenez. Compilé depuis la documentation officielle.

Comment ça marche : VibeLab n'embarque aucune IA et n'envoie rien nulle part — votre progression vit dans ce navigateur (localStorage). Les missions et les commandes produisent des prompts que vous exécutez vous-même dans Claude (claude.ai) ou Claude Code. Le Codex est compilé depuis la documentation officielle Anthropic ; l'outillage évolue vite — en cas de doute, /help dans Claude Code et les release notes font foi.
